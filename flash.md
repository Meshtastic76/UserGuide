# Установка прошивки

Для начала работы, многие устройства потребуется прошить программным обеспечением meshtastic (не бойтесь, это не сложно)

Сществует несколько путей, самый простой это прошивка прямо из браузера - WEB flasher (ВЕБ флешер):
- [Родной прошивальщик meshtastic](https://flasher.meshtastic.org/)
- [Аналог с поддержкой кириллицы](https://mrekin.duckdns.org/flasher/)

Многие устройства для корректной прошивки требуют установки драйвера. Если ваше устройство не отображается в WebFlasher — установите драйвер для вашего контроллера. [Список драйверов](https://wiki.meshworks.ru/node-setup/serial-drivers/)

Подключите устройство к компьютеру через USB-кабель.
1. Откройте сайт веб флешера (ссылки выше)
2. Выберите модель своей платы из списка.
3. Нажмите "Connect", предоставьте доступ к USB-устройству.
4. Выберите тип установки:
  - Update — обновление без удаления данных
  - Full Erase and Install — полная очистка и новая установка
5. Нажмите "Flash" и дождитесь окончания процесса.

После прошивки устройство перезапустится автоматически. Если вы прошиваете первое ваше устройство - устанавливайте стабильную версию прошивки
